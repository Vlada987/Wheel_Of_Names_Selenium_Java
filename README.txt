README for Test Suite

Overview
This is an automated test suite for the Wheel of Names app using Selenium WebDriver 
and TestNG. It supports parallel execution on Windows 7 and Windows 10 with Chrome 
and Firefox browsers, both locally and remotely via Sauce Labs.

 Prerequisites
- Java 8 or higher
- Maven or Gradle
- Selenium WebDriver and TestNG
- Sauce Labs account (for remote execution)
- ChromeDriver and GeckoDriver for local tests

Features
- Page title and description validation
- Name management: Add, shuffle, sort
- Customization: Set spin time and message
- Functional testing
- Save and load functionality
- Localization: Verify different languages

Reporting
Test results are saved in the TestNG output folder. HTML or XML reports can be generated.
